# My Spa Pro Static Website

This is a GitHub-ready static website for My Spa Pro, LLC.

## Pages included

- `index.html` — public marketing homepage
- `privacy.html` — privacy policy for Apple App Store and Google Play
- `terms.html` — terms of service
- `account-deletion.html` — public account/data deletion request page for Google Play and Apple review expectations
- `support.html` — app support URL
- `safety.html` — reporting and user safety page
- `styles.css` — shared styling
- `assets/logo.svg` — editable placeholder logo

## Before publishing

1. Replace `support@myspapro.app` with the real support email.
2. Replace `https://your-domain.com/...` with the final public URLs.
3. Update Privacy Policy sections to match the actual SDKs and data collection in the app.
4. Update payment/subscription language based on whether the app uses Apple IAP, Google Play Billing, Stripe, or no payments.
5. Replace the placeholder logo if you have final brand assets.

- Vercel for GitHub-connected static hosting.

## GitHub Pages quick start

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to Repository Settings > Pages.
4. Set source to Deploy from branch.
5. Select `main` branch and `/root`.
6. Save and wait for GitHub Pages to publish.
